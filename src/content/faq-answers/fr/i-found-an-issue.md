**Merci de l'avoir repéré - signalez-le pour que nous puissions le corriger.**

Le plus rapide est d'ouvrir un rapport de bug sur notre [tracker d'issues GitHub](https://github.com/peltmonger/stardrive/issues). Incluez les étapes pour le reproduire, ce que vous attendiez et ce qui s'est réellement passé. Un extrait de code minimal ou une capture aide énormément.

Si vous n'êtes pas sûr qu'il s'agisse d'un bug ou d'un comportement voulu, le [tableau des discussions](https://github.com/peltmonger/stardrive/discussions) est un bon endroit pour poser la question d'abord.

**Les pull requests avec un correctif sont toujours bienvenues - jetez un œil aux guides de contribution avant de commencer.**
