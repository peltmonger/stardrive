**Pas forcément - mais une compréhension de base du fonctionnement d'un site web aide vraiment.**

Stardrive est conçu pour qu'un assistant de code IA puisse faire la plus grosse partie du travail. Si vous savez décrire ce que vous voulez et relire les changements faits par un agent, vous pouvez aller très loin sans écrire de code vous-même.

Cela dit, des choses comme déployer sur Cloudflare, éditer des fichiers de configuration ou déboguer une erreur de build inattendue seront bien plus fluides si vous (ou quelqu'un dans votre équipe) êtes à l'aise avec les bases de Git, Node.js et de la ligne de commande.

**En bref : c'est un boilerplate de niveau développeur, suffisamment accessible pour des non-développeurs travaillant avec un assistant IA. Mais ce n'est pas un outil no-code !**
