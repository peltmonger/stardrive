**Thanks for catching it - please report it so we can fix it.**

The fastest way is to open a bug report on our [GitHub issue tracker](https://github.com/peltmonger/stardrive/issues). Include the steps to reproduce, what you expected, and what actually happened. A minimal code snippet or screenshot helps a lot.

If you are not sure whether something is a bug or intended behavior, the [discussions board](https://github.com/peltmonger/stardrive/discussions) is a good place to ask first.

**Pull requests with a fix are always welcome - check the contributing guidelines before you start.**
